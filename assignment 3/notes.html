MY NOTES

  • br :The <br> tag inserts a single line break. The br tag is an empty tag which means that it has no end tag.
  • H1 :The <h1> tag is an HTML heading that's most commonly used to mark up a web page title.
  • img :The <img> tag is used to embed an image in an HTML page.
  • p :The <p> HTML element represents a paragraph.
  • b :The <b> tag is used to bold the text in HTML.
